# Thread Context

When triaging an email that is a reply (has in_reply_to):

1. Consider the subject line prefix (Re:, Fwd:) for context
2. The quoted text at the bottom provides conversation history
3. Focus classification on the NEW content (not the quoted portions)
4. If a thread escalates in urgency (e.g., follow-up asking "did you see my email?"), escalate
5. Thread emails from the same sender within the batch should be analyzed together
